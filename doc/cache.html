a boingboing cache with element

an entire library of content, free for the plucking. as a member of the slow data movement, this is the kind of thing i want in my personal search index, along with other tweets/emails/chat-logs and ting

nokogiri subtly failed, returning a partial parse but no warnings or errors. so, i've parsed with Hpricot

require 'hpricot' 
require 'element/Th' 
class E
  Hpricot(E['/bb.xml'].r).search('row').map{|x| 
    c = x.children
    u = c[1].inner_text[0..-6]              ;puts u
    e = u.E
    {u => {'uri' => u, 
        Date => Time.parse(c[3].inner_text).iso8601, 
        Creator =>  c[7].inner_text, 
        Title   =>  c[9].inner_text, 
        Content => c[11].inner_text, 
        '/cat'  => c[17].inner_text.split(/,/) 
      }}.do{|g| 
      e.em.w g, true 
      e.roonga 'boingboing',g.text } rescue nil} 
end

voila, each post is a json graph stored at its path
